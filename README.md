The usage steps of this paper are as follows: 
Step 1: Generate graph-structured data: construct intra-class dissimilarity and similarity graphs for labeled samples, and graph topologies for the augmented set (training set, test set, and nearest neighbors of test samples).
Step 2: Train the model via contrastive learning, then evaluate on the test set using the primary classification branch.
Step 3: Rectify labels of target samples based on local consistency.
